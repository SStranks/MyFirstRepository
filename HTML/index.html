Accessibility

<tabindex> Used to set the tabbing order of focusable elements.

  1. Avoid using positive tabindex to the extent possible.
  2. Use tabindex="0" only for non focusable elements to receive focus.
  3. Use negative tabindex to move focus to elements that have focus set using scripts. Never use negative tabindex for the elements that need to be navigated and are actionable.

Tip: Don't have links open in new window/tab, unless stated in the link title.

Tip: Use href="#id" links in the <header> to direct user straight to important page content, without rendering the links visible.

Tip: In forms use the 'for' attribute in <label> in conjunction with an 'id' attribute in the <input>. This makes the label clickable and places focus on the form field.

Tip: To automatically focus on an input element on page load; use attribute 'autofocus'. e.g. search engine field
